Оболочка ОС Android (или Android-подобная ОС) — операционная система на базе Android. Чаще всего разрабатываются разработчиками мобильных устройств для своих же продуктов. Отличаются не только графическим интерфейсом, но и расширенной/изменённой функциональностью.

Исходный код

В болшинстве случаев, код таких систем открыт.

Примеры действующих оболочек Android

  • OxygenOS — разработана компанией OnePlus для собственных устройств
  • ColorOS — разработана компанией OPPO для собственных устройств
  • MIUI — разработана компанией Xiaomi для собственных устройств
  • FlymeOS — разработана компанией Meizu для собственных устройств
  • FireOS — разработана компанией Amazon для собственных устройств (Amazon Fire)
  • EMUI — разработана компанией Huawei для собственных устройств
  • One UI — разработана компанией Samsung для собственных устройств
